# Python Voice Assistant

A simple voice assistant built using Python that listens to voice commands and performs tasks like playing music, searching Google, telling the time, and retrieving information from Wikipedia.

## Features

* Voice command recognition
* Text-to-speech responses
* Play songs on YouTube
* Search Google
* Wikipedia information lookup
* Tell current time
* Interactive voice responses

## Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* pywhatkit
* Wikipedia API

## Installation

1. Clone the repository:

git clone https://github.com/yourusername/python-voice-assistant.git

2. Navigate to the project folder:

cd python-voice-assistant

3. Install required libraries:

pip install -r requirements.txt

## Usage

Run the assistant:

python jarvis.py

Example commands:

* "Alexa play Shape of You"
* "Alexa search for Python programming"
* "Alexa who is Elon Musk"
* "Alexa what is the time"
* "Alexa close"
